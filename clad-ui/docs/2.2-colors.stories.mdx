# Colors

It is encouraged to use **semantic** color names instead of the primitive tokens. Please help expand the **semantic colors** where they make sense.

```js
import { css } from 'linaria';
import { sx } from 'clad-ui/utils';
import theme from '@clad-ui/theme';

// destructured object shortcuts
const { colors } = theme;

const buttonStyle = css`
  background-color: ${colors.accent};
`;

// theme-aware helper
const swatchStyle = css`
  ${sx({
    color: 'text',
  })}
`;
```

## Full Color Pallete

For each color tone, the main color should be **#2** (e.g. **red2**, **orange2**, **yellow2**, etc...). **#1** are usually for pressed down state, **#3** for hover states and **#4, #5** for backgrounds...
