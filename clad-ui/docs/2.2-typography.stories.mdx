# Typography

Typography tokens are defined in following theme's fields:

- `theme.fonts`: Font stack
- `theme.fontSizes`: Font sizes
- `theme.fontWeights`: Font weights
- `theme.lineHeights`: Line heights

```js
import { styled } from 'linaria/react';
import theme from '@clad-ui/theme';

const Heading1 = styled.h1`
  font-size: ${theme.fontSizes.h1};
  font-weight: ${theme.fontWeights.bold};
  color: ${theme.colors.text};
`;
```

## Font stack

```
theme.fonts.[body|heading|monospace]
```

## Font sizes

```
theme.fontSizes.[xxs | xs | sm | md | lg | xl | xxl | xxxl ];
// aliases:
theme.fontSizes.[    body | h6 | h5 | h4 | h3 | h2  | h1   ];
```
